# 🧠 Teste de Sanidade Dev

Um pequeno projeto criado apenas para **testar a sanidade mental dos desenvolvedores** 😅  
A ideia é simples: enviar para amigos ou grupos de programadores e ver quem consegue resolver (ou quem enlouquece primeiro).

Projeto criado por diversão e para exercitar lógica, interação e criatividade.

---

## 🚀 Demonstração

Se você recebeu esse projeto, provavelmente alguém quer testar sua sanidade como dev.

Boa sorte. Você vai precisar.

---

## 🧩 Como funciona

O projeto apresenta um pequeno desafio/interação onde o usuário precisa:

- Interpretar o problema
- Pensar um pouco fora do padrão
- Resolver o desafio proposto

A graça está em **ver as reações das pessoas tentando descobrir a solução**.

## ▶️ Como executar o projeto

1. Clone o repositório


git clone https://github.com/LeandroDevLab/teste-sanidade-dev.git


2. Entre na pasta do projeto


cd teste-sanidade-dev


3. Abra o arquivo `index.html` no navegador.

Pronto 🎉
